# 魔石インフレーションRPG Rebuild v2.4

## 主な変更

- v2.3を土台に、ビルドを組む楽しさが出る装備効果を追加
- 装備効果を拡張
  - 会心率
  - 技成長速度
  - 魔石追加
  - 勝利時MP回復
  - 被ダメージ軽減
  - ボス特効
  - 既存のレア率、経験値、ダメージ、複数攻撃、吸収効率も維持
- 後半の経験値不足を修正
  - 敵のHP/攻撃/防御/Lv/レア/ボスに応じて経験値が増える
  - レベル必要経験値の伸びを緩和
- 敵を全体的に強化
  - 後半ステージほどHP/攻撃/防御が上がる
  - レア敵/ラスボスはさらに強化
- 魔石吸収を個別閾値方式へ変更
  - 各魔石ごとにHP/MP/攻撃/防御の閾値を持つ
  - 閾値を超えると上昇量が段階的に下がる
  - 0にはならない
- 敵からの技取得はいったん廃止
  - デフォルト技3つを使い込んで強化する方針へ変更
  - 育成画面でデフォルト技Lvと技EXPを確認可能
