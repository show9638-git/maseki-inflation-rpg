# 魔石転生サバイバー v0.3

## v0.3修正内容

- 慣性をなくし、離した瞬間に停止
- 操作入力の滑りを軽減
- 種族を「人族・獣族・竜族」に固定
- レベルアップ候補を「共通スキル + 選択種族スキル」のみに変更
- 敵撃破時にGold獲得
- Goldは死亡しても減らない永続通貨
- Goldガチャ追加
- ガチャで装備とキャラクタースキンを獲得
- かぶりは +値として蓄積
- +10 / +100 / +1000 / +10000 で性能補正が段階強化
- 所持品画面を追加

## 操作

- スマホ：画面のどこでも押してドラッグ。離すと即停止。
- PC：WASD / 矢印キー
