# 魔石転生サバイバー v0.8

## v0.8の作り直し内容

- 1枚絵背景を廃止
- タイル式の天空神殿マップに変更
- 床 / 壁 / 建造物 / 柱 / 階段 / 装飾床を分離
- 壁、建造物、柱、外周に当たり判定を実装
- 障害物オブジェクトは廃止
- 人族をスプライトシート式ドットキャラに変更
- 人族に歩行4フレームアニメを実装
- 人族の進化段階ごとに見た目を変更
- スキルアイコンと演出をドットエフェクト寄りに変更
- 敵のインフレをさらに強化
- Gold/転生/ガチャ/アイテムの基本要素は継続
