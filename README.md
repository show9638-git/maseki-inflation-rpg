# 魔石インフレーションRPG Rebuild v1.2

## 変更内容

- 回復薬ボタン削除
- 10%吸収ボタン削除
- 装備を30個追加
- 仲間1/仲間2を追加・育成/装備対象に対応
- 通常の戦闘画面から主人公/仲間1/仲間2の対象切替が可能
- 主人公/仲間1/仲間2のイラストをユーザー側で変更可能
- フィーバー時間を約1分に延長
- フィーバーゲージは以前よりたまりにくく調整
- 一括吸収でも特技習得判定が発生
- BGMを3種類に増加
- 効果音パターンを追加
- 場所ごとに敵出現数倍率を1倍〜3倍まで変更可能
- 最大同時表示は3体のまま安定化
