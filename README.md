# 魔石インフレーションRPG Rebuild v1

作り直し版です。

## 主な仕様

- ドラクエ風の正面バトル画面
- 敵イラストは1体ずつカード表示
- 複数エネミー時は最大3体分を同時表示
- エリアごとに通常敵とレア敵を追加
- レア敵は各エリア3体
- 仲間は白黒虹の女性1人
- 仲間も主人公と同様に育成/装備対象を切り替え可能
- フィーバーゲージ追加
- 敵撃破でフィーバー蓄積
- フィーバー満タンで一定時間、自動で高速撃破
- 魔石吸収は総吸収量に応じて成長効率が下がる
- 攻撃、特技、一括吸収は長押し対応
- GitHub Pagesでそのまま公開可能
