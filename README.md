# 魔石インフレーションRPG v0.2

## 追加内容

- BGM/効果音を追加
  - 外部音源ファイルではなく、ブラウザの Web Audio API で鳴らしています。
  - 画面右上の「音ON」で開始します。
- 主人公・敵・アイコン素材シートを追加
- 主人公画像差し替え機能を追加
- 魔石一括吸収ボタンを追加
- 魔石ドロップ仕様を変更
  - 敵1体につき基本1個
  - まれに複数個
- 習得技を9枠までセット可能
- 技のつけ外し可能
- 敵を20体以上に拡張

## GitHub Pagesへ反映する時

このフォルダの中身を既存リポジトリへ上書きしてから以下を実行します。

```powershell
git add .
git commit -m "Update RPG v0.2 assets audio skills"
git push
```

GitHub Pages反映には数十秒〜数分かかることがあります。
