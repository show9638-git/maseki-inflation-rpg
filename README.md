# 魔石インフレーションRPG Rebuild v2

読み込みエラー対策のため、機能を引き継いだうえで再構築した安定版です。

## 引き継ぎ機能

- 回復薬削除
- 10%吸収削除
- 一括吸収
- 一括吸収時の特技習得判定
- 主人公/仲間1/仲間2
- 通常戦闘画面から育成/装備対象の切替
- 主人公/仲間1/仲間2の画像差し替え
- 装備40個
- レア率/経験値/ダメージ/複数攻撃/吸収効率/技習得率付き装備
- 場所ごとの敵出現数倍率 1〜3倍
- 最大3体同時表示
- フィーバー約1分
- BGM 3種類
- 効果音
- 長押し対応
- 敵/レア敵/敵絵一覧

## 反映

```powershell
git add .
git commit -m "Rebuild v2 stable"
git push
```
