# 魔石転生サバイバー v0.12

## v0.12 修正内容

### マップ修正

- 黒い虚空タイルが画面内に大きく出る問題を修正
- マップ全体を神殿床ベースに変更
- 先ほど生成した天空神殿タイルの方向性に合わせ、ゲーム用タイルを再生成
- マス目が目立たないよう、床タイルの強い区切り線を排除
- 壁 / 建造物 / 柱 / 階段 / クリスタル床を見分けやすく修正
- 当たり判定は以下に適用
  - 壁
  - 建造物
  - 柱
  - マップ外周

### アバター修正

- アバターの前後姿が交互に出る問題を修正
- 方向別スプライトシートを新規生成
- 移動方向に応じて使用フレームを固定
  - 下移動：前向きフレーム
  - 上移動：後ろ向きフレーム
  - 左移動：左向きフレーム
  - 右移動：右向きフレーム
- 進化段階は方向フレームを壊さず、オーラや翼を足す方式に変更

### 敵修正

- 敵にも壁 / 建造物 / 柱 / 外周の当たり判定を適用
- 幽霊系の敵だけ壁抜け可能
- 敵スプライトは3フレームアニメとして使用
