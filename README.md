- ドット絵マップを追加

- アイテム20種類を実装

  - 回復薬
  - 秘薬
  - 強化薬
  - 秘伝書
  - 成長薬
  - 磁石
  - 巨大化の種
  - ほか12種
- 地図取得で右上にミニマップ表示
- ミニマップにアイテム位置を表示
- 人族の通常ドット + 進化ドットを描画
- 各スキルのドット風アイコンを描画
- v0.4までの
  - 敵インフレ強化
  - スキル40種
  - 3秒レベルアップ選択
  - ガチャ500種
  を引き継ぎ

## v0.7修正内容

- 障害物の配置をマップ固定に変更
- 障害物の数を削減
- アイテム生成速度を低下
- 画面内に出るアイテム数の上限を削減
- ガチャ装備ごとに効果を付与
  - N：ステータス小強化
  - R：ステータス中強化
  - SR：ステータス大強化
  - SSR：特殊効果
  - UR：超特殊効果
- +10 / +100 / +1000 / +10000 のボーナスを大幅強化
- 敵のHP、攻撃、速度、出現数のインフレを強化
- Gold獲得量を低下
- 転生強化にGold取得率を追加
- 転生強化の上限を最大1000へ拡張
- 転生強化の上昇率をレベルが上がるほど伸びる方式へ変更
