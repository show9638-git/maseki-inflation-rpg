# 魔石転生サバイバー v0.2

## v0.2修正内容

- 種族を「人族・獣族・竜族」の3種類に整理
- 見た目は後からスキン追加できる想定にし、種族とスキンを分離しやすい構造へ整理
- 初期移動速度を大幅アップ
- 入力レスポンス改善
- 画面内のどこを押しても移動できる方式に変更
- スティックボタン廃止
- BGM追加
- 効果音追加
- スマホ操作優先で調整

## 操作

- スマホ：画面のどこでも押して、動きたい方向へドラッグ
- PC：WASD / 矢印キー
