# 魔石転生サバイバー v0.1

ヴァンサバ風のアクション試作版です。

## 入っている要素

- 見下ろし型アクション
- スマホ仮想スティック
- PCキーボード移動 WASD / 矢印
- 自動攻撃
- 敵の無限湧き
- 敵が時間経過で強化
- 経験値回収
- レベルアップ時のスキル選択
- 種族 6種
- 種族ごとの親和スキル
- スキル進化
- レベルによる見た目進化
- ボス出現
- 死亡リザルト
- ハイスコア保存
- 転生ポイント
- 転生強化
- localStorage保存

## 反映コマンド

PowerShellでリポジトリに上書き後:

```powershell
git add .
git commit -m "Add maseki tensei survivor v0.1"
git push
```

古い表示が残る場合:

```text
?v=survivor01
```
