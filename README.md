# 魔石転生サバイバー v0.13

## v0.13 修正内容

### 生成素材のゲーム反映

- 生成済みの完成マップ素材 `天空の祭壇と光の広場.png` をゲーム内背景として反映
- 生成済みの主人公素材 `歩行サイクルのチビキャラクター.png` をゲーム内スプライトとして反映
- 敵素材は既存のアニメーションスプライトを継続利用

### マップ修正

- これまでのタイル分割マップを廃止
- 生成済みの綺麗な天空神殿マップをそのまま背景として表示
- 神殿床・通路に合わせた当たり判定を作成
- 大型建造物 / 柱 / 中央台座 / 外周には当たり判定を設定
- 黒いタイルや謎の床が出る問題を回避

### アバター修正

- 主人公スプライトを4方向×4フレームで再構築
- 下移動：前向き
- 上移動：後ろ向き
- 左移動：左向き
- 右移動：右向き
- 前後左右の素材が混ざる問題を修正
- 見切れを抑えるため、各フレームを中央寄せで再生成

### 難易度修正

- 最序盤の敵HP / 攻撃力 / 出現速度を低下
- プレイヤー初期HPと攻撃力を上方修正
- 1分以降から敵が強くインフレするよう調整
- 序盤に即死しにくく、スキル確認に進めるよう調整

### 敵当たり判定

- 通常敵は神殿外周 / 建造物 / 柱にぶつかる
- 幽霊系の敵だけ壁抜け可能
