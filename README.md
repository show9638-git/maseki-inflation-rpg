# 魔石転生サバイバー v0.4

## v0.4修正内容

- 敵のインフレを強化
  - 時間経過でHP/攻撃/移動速度/出現数がより強く増加
- レベルアップ時のスキル選択に3秒制限を追加
  - 未選択の場合、選択肢からランダム取得
- 共通スキルを10個実装
- 人族スキルを10個実装
- 獣族スキルを10個実装
- 竜族スキルを10個実装
- レベルアップ候補は「共通 + 選択種族」のみ
- ガチャ中身を500個実装
- ガチャ確率
  - スキン 0.1%
  - UR 0.5%
  - SSR 1%
  - SR 10%
  - R 20%
  - N 残り
- 100連ガチャ追加
