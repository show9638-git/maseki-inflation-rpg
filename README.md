# 魔石転生サバイバー v0.5

## v0.5追加内容

- ドット絵マップを追加
- 障害物（木 / 岩 / 遺跡）を追加
- 移動しやすさを少し制限
- ランダムタイミングでランダムアイテムがマップ各地に出現
- アイテム20種類を実装
  - 地図
  - 回復薬
  - 秘薬
  - 強化薬
  - 秘伝書
  - 成長薬
  - 磁石
  - 巨大化の種
  - ほか12種
- 地図取得で右上にミニマップ表示
- ミニマップにアイテム位置を表示
- 人族の通常ドット + 進化ドットを描画
- 各スキルのドット風アイコンを描画
- v0.4までの
  - 敵インフレ強化
  - スキル40種
  - 3秒レベルアップ選択
  - ガチャ500種
  を引き継ぎ
