# 魔石インフレーションRPG Mobile

スマホで遊びやすいように作り直した版です。

## 遊び方

1. ZIPを解凍
2. `index.html` を開く
3. スマホの場合は、ファイルをスマホに送ってブラウザで開く

## スマホ対応内容

- タブ式UI
- ボタンを大きく調整
- 画面幅に応じて1列/2列表示
- 下部にコマンドボタン固定
- セーブ/ロード対応
- タッチ操作向けCSS

## 安定してスマホで開く方法

PCでこのフォルダを開き、PowerShellで以下を実行します。

```powershell
python -m http.server 8000
```

その後、スマホから `http://PCのIPアドレス:8000` にアクセスします。
