# 魔石インフレーションRPG Rebuild v3.1

## 作り込み強化アップデート

v3.0を土台に、見た目・操作感・敵の手応え・装備確認・進捗報酬を強化した版です。

## 主な変更

- 主人公/仲間の動くドットを12フレーム化
  - 待機3フレーム
  - 攻撃4フレーム
  - 特技4フレーム
  - 被弾1フレーム
- 敵ドット絵を全288体分、個体ごとに再生成
  - 種族別の骨格
  - ステージ別カラー
  - レア/メタル/ボス用の装飾
  - 個体IDごとの追加装飾
- 全196装備にドットアイコンを追加
  - `assets/items/`
- 装備画面を改修
  - 今装備しているものを上部に表示
  - 装備中カードを強調表示
  - 装備アイコン表示
- UIを改修
  - 色分け
  - 柄・発光
  - タブ/ボタン/カードの視認性向上
- オート速度を5倍相当に変更
- ステージごとのBGMへ変更
  - WebAudio多層BGM
  - メロディ、ベース、簡易パーカッション
- 敵をさらに強化
  - 最初の草原は控えめ
  - 後半ステージほど大幅強化
  - レア/ボス/メタルはさらに強化
- 後半経験値不足を緩和
  - 強い敵ほど経験値増加
  - メタル系の経験値を大幅増加
  - 装備/進捗報酬の経験値効果を強化
- ステージ進捗をさらに貯まりにくく調整
- 進捗100%報酬を強化
  - 経験値
  - 火力
  - 吸収効率
  - レア率
  - ドロップ
  - 技成長
  - メタル率
  - 被ダメ軽減
