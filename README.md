# 魔石インフレーションRPG Rebuild v2.3

## 主な変更

- 主人公/仲間1のデフォルトドット画像を同梱
- 画像枠に収まるよう、128x128の内蔵画像を使用
- 画像アップロード時も128x128へ自動縮小して保存
- オート速度を3倍相当に変更
- 魔石吸収時のステータス上昇量を強化
- 敵ごとの個別魔石に変更
- 強い敵/レア敵/ラスボスの魔石ほど吸収力・売却額が高い
- 魔石売却も吸収と同じくプルダウン選択式
- 取得可能技を敵の数に合わせて追加
- 攻撃以外の技を追加
  - 回復
  - バフ
  - シールド
  - ドレイン
  - 魔石取得
  - 経験値取得
  - MP回復
- 技レベルアップ要素を復活
- 装備に装備枠を記載
- 敵出現数倍率を最大5倍へ変更
- 仲間1を主人公に近いステータスまで比例成長
- 主人公デフォルト技を最強寄りに変更
  - 単体最強威力
  - 全体最強威力
  - ランダム10回最強威力
- 仲間1の技を変更
  - 単体最強威力
  - 全体最強威力
  - 全体最強回復
- ステージを10個追加
- ラスボス「終焉神ラグナ・ゼロ」を追加
- レア敵からレア技、レア装備、レア素材を取得可能
